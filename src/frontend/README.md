# Training Tracker Frontend

## Requisitos
- Node 18+
- Backend rodando em http://localhost:3001 (configurável via VITE_API_URL)

## Setup
1. Crie `.env` baseado em `.env.example` e ajuste `VITE_API_URL` se necessário.
2. Instale dependências:
   ```bash
   npm install
   ```

## Executar
- Desenvolvimento:
  ```bash
  npm run dev
  ```
- Build e preview:
  ```bash
  npm run build && npm run preview
  ```
